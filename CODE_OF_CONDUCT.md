# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc-2.1], a
widely-used community code of conduct for open-source projects.

The full text and all version history are available at:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this community (issues, pull requests, discussions,
or any other project space) you agree to follow it.

## Scope

The Code of Conduct applies in all project spaces, and also when an
individual is officially representing the project in public spaces.

## Enforcement

Reports of behavior that violates the Code of Conduct may be sent to the
project maintainer privately through GitHub Security Advisories:

  https://github.com/manop55555/roe/security/advisories/new

GitHub Security Advisories is used here because it provides a private
channel that the project already supports. There is no separate email
contact.

All reports are reviewed and investigated promptly and fairly. Reporter
identity is kept confidential.
